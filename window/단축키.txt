Ctrl+W | 창 닫기

 f2 | 이름 바꾸기

 Win+E | 파일 탐색기

 Win+좌우 | 창 좌우분할

 Win+상하 | 창 상하분할(창 크기 조절)

 Win+D | 바탕화면 보기 & 다시 복원

 가상데스크톱
데스크톱 확인 | Win+Tab
데스크톱 이동 | Ctrl+Win+좌우

2:47 작업표시줄 프로그램 열기 | Win+숫자
(새창으로 열기 Win+Shift+숫자)

 윈도우 설정 | Win+i

 블록 설정 | Ctrl+l

 화면 잠금 | Win+l (카톡잠금 Crtl+l)

 화면 확대/축소 | Ctrl+휠 (되돌리기 Crtl+0)

 화면 캡쳐 | Win+Shift+s


ctrl + W -> 창 닫기

f2 -> 이름 바꾸기

win + E -> 파일탐색기 
win + 좌우방향키 -> 좌우분할 
- win + 상하방향키 -> 상하분할
- or 최대화/이전크기로 복원/최소화
win + D -> 바탕화면 보기 (2번 누르면 복원가능)
win + M  -> 창 전체 최소화 (2번 누르면 복원이 안됨)
- win + shift + M -> 복원단축기 
ctrl + win + D -> 가상 데스크톱 만들기 
ctrl + win +f4 -> 가상 데스크톱 삭제 
- ctrl + win + 좌우방향키 -> 가상 데스크톱 이동
- win + tab -> 만들어놓은 데스크톱 확인, 데스크톱을 추가하거나 삭제 가능
win + 숫자키 (1~0) -> 작업표시줄에 있는 프로그램 열기 
- win + shift + 숫자키 (1~0) -> 새 창 열기 
win + T -> 작업 표시줄에 있는 앱을 직접 선택 
win + I -> 윈도우즈 설정
shift + 방향키 -> 텍스트를 블록으로 설정  (방향키를 여러번 눌러야 함)
ctrl + shift + 방향키 -> 텍스트를 블록으로 설정 (단어단위로 설정됨)
win + L -> 윈도우 잠금 
- ctrl + L -> 카톡 잠금
shift + enter -> PC 카톡 줄바꿈 
ctrl + 마우스 휠 -> 화면 확대/축소
- ctrl + 0 -> 배율 100%
 PRTSC 누르기 -> 그림판 열기 -> 원하는 부분 자르기 -> 다시 그림파일로 저장  
= win + shift + S -> 화면 캡쳐 (선택한 부분은 클립보드에 저장됨, 필요한 곳에서 ctrl + V 하면 끝. 파일 저장도 가능함)
